# Used Car Price Analysis

Data cleaning and preprocessing of the **UCI Automobile dataset** using Python.

## Project Overview
This project focuses on preparing the dataset for further analysis and machine learning by performing:

- data cleaning
- handling missing values
- data type correction
- normalization
- feature binning
- basic visualization

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

## Dataset
UCI Machine Learning Repository – Automobile Dataset

https://archive.ics.uci.edu/ml/datasets/automobile
